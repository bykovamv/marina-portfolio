# Марина — Персональная визитка-портфолио

> 🎓 **Учебный проект** по веб-разработке

## Описание

Одностраничный сайт-визитка, созданный в учебных целях для изучения современных технологий вёрстки и веб-дизайна.

## 🚀 Технологии

- **HTML5** — семантическая разметка
- **CSS3** — стилизация и анимации
  - Градиентный фон
  - Анимация появления текста (typewriter effect)
  - 3D-переворот карточек при наведении
  - Анимация падающих элементов
  - Адаптивная вёрстка (mobile-first)
- **JavaScript** — интерактивность
  - Intersection Observer API для анимации при скролле
  - Динамическая генерация элементов (падающие листы)

## ✨ Особенности

- 🌟 **Анимация имени** — буквы появляются по очереди
- 🎴 **Карточки с переворотом** — 3D-эффект при наведении
- 📱 **Адаптивный дизайн** — корректно отображается на всех устройствах
- 🍃 **Падающие листы** — фоновая анимация
- 🎨 **Светло-фиолетовая тема** — современный минималистичный дизайн
